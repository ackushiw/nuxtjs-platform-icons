## Setup
- Add `@nuxtjs/platform-icons` dependency using yarn or npm to your project
- Add `@nuxtjs/platform-icons` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    '@nuxtjs/platform-icons',

    // With options
    ['@nuxtjs/platform-icons', { /* module options */ }],
 ]
}
```
